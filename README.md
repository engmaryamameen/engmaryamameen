## ⭐ Featured Project

###  [TaskForge](https://github.com/engmaryamameen/TaskForge)

TaskForge includes organization-based project management, real-time collaboration, secure authentication, billing, email notifications, and a polished dashboard UI.

**Highlights**

- 🏢 Multi-tenant organization management with role-based access control
- ✅ Kanban-style project and task management
- 🔔 Real-time in-app notifications using Socket.IO
- 📧 Email notifications with Nodemailer
- 🔐 JWT authentication, refresh token rotation, Argon2 password hashing
- 💳 Stripe billing integration
- ⚡ Redis-backed caching, rate limiting, and BullMQ workers
- 🎯 Polished dashboard UI built for portfolio presentation

**Tech Stack**

`Next.js 15` · `React 19` · `TypeScript` · `Tailwind CSS` · `NestJS` · `PostgreSQL` · `TypeORM` · `Redis` · `Socket.IO` · `BullMQ` · `Stripe` · `Docker`

[Live Demo](https://task-forge-demo.vercel.app/) · [Source Code](https://github.com/engmaryamameen/TaskForge)
